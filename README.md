# Well-Laid Plans in Multiplayer
Well-Laid Plans is a great card for the Silent, but it's banned from multiplayer, presumably due to asking the player to pick cards during the enemy turn.

This mod adds in a multiplayer-safe version of Well-Laid Plans when playing multiplayer.

## Installation
Unzip the contents of the .zip into the `mods` folder in the same location as `sts2.exe`.
Requires [BaseLib](https://github.com/Alchyr/BaseLib-StS2/releases/latest) and at least v0.105.1 of STS2 (beta branch at the time of writing).
